# GitWise

**AI-Powered Git Workflow Assistant**

GitWise is a Python-based application designed to help students, educators, and aspiring developers learn and navigate Git and GitHub workflows with confidence. Instead of searching through documentation or guessing at command-line errors, users can receive guided assistance for common Git tasks, workflow explanations, and troubleshooting support.

The goal of GitWise is to reduce the learning curve associated with version control while reinforcing professional Git and GitHub practices used in software development, cybersecurity, DevOps, and IT environments.

---

## Features

### Workflow Guidance

Receive step-by-step assistance for common Git and GitHub tasks, including:

* Initializing a repository
* Cloning repositories
* Creating and switching branches
* Staging and committing changes
* Pushing to GitHub
* Pulling updates
* Merging branches
* Creating pull requests
* Managing repository workflows

### Git Error Diagnosis

Paste Git error messages and receive:

* Plain-language explanations
* Common causes
* Recommended recovery steps
* Best-practice guidance

### Educational Support

Designed with learning in mind:

* Beginner-friendly explanations
* Real-world workflow examples
* Command explanations
* Professional Git terminology

### Future AI Features

Planned enhancements include:

* Natural language Git assistance
* Repository analysis
* Workflow recommendations
* AI-powered troubleshooting
* GitHub Classroom integration
* Instructor and student modes

---

## Why GitWise?

Many new developers struggle with Git because error messages are often confusing and workflow concepts can be difficult to visualize. GitWise serves as a learning companion that helps users understand not only which command to use, but why it should be used.

By focusing on explanation and guidance rather than automation, GitWise helps users build confidence and develop skills that transfer directly into academic and professional environments.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/gitwise.git
cd gitwise
```

Create and activate a virtual environment:

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

macOS/Linux:

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch the application:

```bash
streamlit run app.py
```
